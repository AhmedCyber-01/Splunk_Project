# Splunk SIEM Log Analysis Projects

This repository contains a collection of log analysis projects built using Splunk SIEM. Each project demonstrates how to ingest logs, analyze security events, and create visualizations using SPL (Search Processing Language).

## SSH Log Analysis

Analyze SSH authentication logs to identify:

* Failed login attempts
* Successful logins
* Brute-force attack patterns
* Suspicious source IP addresses
* User authentication activity

## DNS Log Analysis

Analyze DNS logs to:

* Monitor DNS query activity
* Identify frequently requested domains
* Detect suspicious or unusual DNS behavior
* Visualize DNS traffic patterns

## FTP Log Analysis

Analyze FTP logs to:

* Track file transfer activity
* Monitor user access
* Identify abnormal login behavior
* Detect suspicious file transfers

## HTTP Log Analysis

Analyze HTTP logs to:

* Monitor web traffic
* Identify popular resources
* Detect unusual request patterns
* Investigate web-based threats

## Tunnel Log Analysis

Analyze Zeek tunnel logs to:

* Monitor tunnel traffic
* Identify unusual connections
* Detect suspicious tunneling activity
* Visualize tunnel usage

## SMTP Log Analysis

Analyze SMTP logs to:

* Monitor email traffic
* Track sender and recipient activity
* Detect suspicious email behavior
* Investigate potential phishing indicators

## DHCP Log Analysis

Analyze DHCP logs to:

* Track IP address assignments
* Monitor lease activity
* Identify unusual network behavior
* Visualize DHCP traffic trends

## Tools Used

* Splunk Enterprise
* SPL (Search Processing Language)
* Zeek Logs
* Network Security Logs

## Objective

The goal of these projects is to gain hands-on experience with log analysis, threat detection, security monitoring, and dashboard creation using Splunk SIEM.
